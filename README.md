# Harmony AM: Personal Music Analytics

An interactive dashboard for exploring music listening habits, mood patterns, 
and audio features across academic semesters.

**[Live Demo](https://harmonyam.streamlit.app/)**

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red)
![License](https://img.shields.io/badge/License-MIT-green)

## Features

- **Radar Chart** — Compare audio features against global averages
- **Sunburst Chart** — Hierarchical mood sentiment breakdown
- **Horizon Chart** — Time-series mood variance visualization
- **Ridge Plot** — Podcast listening pattern shifts

## Color Psychology

- **Warm oranges** → Positive emotions, energy, happiness
- **Cool blues** → Calm, melancholy, introspection
- **Red accents** → Key metrics and attention

## Quick Start

pip install -r requirements.txt
streamlit run app.py
